## Como executar:

- Fazer o download ZIP;
- Instalar o http-server - No terminal, executar:

npm install --global http-server

- No cmd, ir para a pasta que está o projeto e executar: 

http-server

- No navegador, entrar no endereço:

http://localhost:8080
